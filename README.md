# 🚀 Smart File Organizer (Java)

## 📌 Overview
A smart file organization system built using Java (Swing) that automatically categorizes and organizes files based on type and content.

---

## ✨ Features
- 📂 Select any folder from system
- 🧠 Smart file categorization (type + keyword-based)
- 🔄 Undo feature (restore files to original location)
- 📊 Dashboard (file statistics)
- 🏷️ Automatic file renaming (clean naming)
- ⚡ Real-time file organization

---

## 🛠️ Tech Stack
- Java
- Swing (UI)
- File Handling (Java IO, NIO)

---

## 🧠 How It Works
1. User selects a folder  
2. System scans all files  
3. Files are categorized into:
   - Documents
   - Images
   - Videos
   - Code
   - Resume / Bills (keyword-based)
4. Files are automatically moved into respective folders  
5. Dashboard shows file statistics  
6. Undo option restores original structure

## 🚀 How to Run
1. Open project in VS Code  
2. Run Main.java  
3. Click "Select Folder"  
4. Choose any folder  
5. Files will be organized automatically
